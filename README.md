# 🪖 Мир Танков — Telegram Mini App

Telegram Mini App с коллекцией мини-игр на тему "Мир Танков".

## 📁 Структура проекта

```
├── bot.py                  # Telegram бот (Python)
├── requirements.txt        # Зависимости Python
├── README.md               # Этот файл
└── webapp/                 # Веб-приложение (Mini App)
    ├── index.html          # Главная страница (хаб игр)
    ├── wheel.html          # Колесо Фортуны
    ├── css/
    │   ├── style.css       # Общие стили
    │   └── wheel.css       # Стили Колеса Фортуны
    └── js/
        ├── app.js          # Общая логика
        └── wheel.js        # Логика Колеса Фортуны
```

## 🚀 Как запустить

### 1. Создайте бота в Telegram
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен

### 2. Разместите веб-приложение
Вариант A — **GitHub Pages** (бесплатно):
1. Загрузите папку `webapp` в GitHub репозиторий
2. Включите GitHub Pages в настройках репозитория
3. URL будет: `https://username.github.io/repo-name/`

Вариант B — **ngrok** (для тестирования):
```bash
# В папке webapp запустите простой сервер
python -m http.server 8080
# В другом терминале
ngrok http 8080
```

### 3. Настройте бота
1. Откройте `bot.py`
2. Замените `ВАШ_ТОКЕН_БОТА` на токен от BotFather
3. Замените `ВАШ_URL` на URL вашего веб-приложения

### 4. Запустите бота
```bash
pip install -r requirements.txt
python bot.py
```
